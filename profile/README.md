# Lavandula

> Open-source reasoning tools for life science research.

Lavandula builds tools that help researchers turn raw experimental data into publication-ready insight — combining domain-encoded rules with AI-augmented reasoning, with full auditability throughout.

## Projects

- **[Proteia](https://github.com/lavandula-bio/proteia)** — Western blot analysis (Phase 1 starting Q4 2026)
